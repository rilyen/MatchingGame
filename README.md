# Matching Game
JavaFX application that allows users to play a memory-matching game on a 4x4 grid.
The game involves flipping pairs of cards to find matches. 

# Build and Run the Application
This project uses Java 22 and JavaFX SDK version 22.0.1. Ensure the dependencies are loaded from Maven and SDK is added to library.
Build and run the main file found in
``src/ca.cmpt213.asn4.memorygame``

# Playing the Game
Click on a card to flip it. 1 move consists of flipping 2 cards.
The objective is to find all the matching pairs with the fewest number of tries. 
The game ends when all pairs have been matched
